Database

This application is configured to use a PostgreSQL database for development and production but to test against an inmemory database

Postgres settings can be found in the /conf/application.conf. So you will either have to create this database upfront or change these setting

  
    slick.dbs.default.driver="slick.driver.PostgresDriver$"
    slick.dbs.default.db.driver="org.postgresql.Driver"
    slick.dbs.default.db.url="jdbc:postgresql://localhost/swengs"
    slick.dbs.default.db.user="swengs"
    slick.dbs.default.db.password="swengs"

The test-database is configured as part of the trait helpers.SecurityTestContext.
